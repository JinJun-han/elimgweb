### 2️⃣ **뉴스 슬라이드쇼 기능 구현**

#### **기술 구현:**

**State 추가:**
```javascript
const [newsSlideIndex, setNewsSlideIndex] = useState(0);
```

**자동 전환 로직:**
```javascript
// 네팔반 뉴스 슬라이드쇼 (3초마다 자동 전환)
useEffect(() => {
  const newsSlideInterval = setInterval(() => {
    setNewsSlideIndex(prev => (prev + 1) % (NEWS[0].images?.length || 1));
  }, 3000);
  return () => clearInterval(newsSlideInterval);
}, []);
```

**렌더링 로직:**
```javascript
{n.images ? (
  <>
    {/* 슬라이드 이미지 */}
    {n.images.map((img, idx) => (
      <div key={idx} style={{
        position: 'absolute',
        inset: 0,
        backgroundImage: `url(${img})`,
        backgroundSize: 'cover',
        backgroundPosition: 'center',
        opacity: newsSlideIndex === idx ? 1 : 0,
        transition: 'opacity 1s ease-in-out'
      }}/>
    ))}
    
    {/* 슬라이드 인디케이터 */}
    <div style={{
      position: 'absolute',
      bottom: 10,
      left: '50%',
      transform: 'translateX(-50%)',
      display: 'flex',
      gap: 6,
      zIndex: 2
    }}>
      {n.images.map((_, idx) => (
        <div key={idx} style={{
          width: 8,
          height: 8,
          borderRadius: '50%',
          background: newsSlideIndex === idx ? '#fff' : 'rgba(255,255,255,.3)',
          transition: 'all .3s'
        }}/>
      ))}
    </div>
  </>
) : (
  <img src={n.img} style={{...}} alt={n.title}/>
)}
```

#### **기능 특징:**
- ⏱️ **전환 간격**: 3초 (히어로는 5초)
- 🎨 **전환 효과**: 1초 페이드 인/아웃
- 📍 **인디케이터**: 하단 중앙, 현재 슬라이드 흰색 강조
- 🔄 **자동 순환**: 10번째 → 1번째로 자동 반복
- 📱 **반응형**: 모바일에서도 완벽 작동

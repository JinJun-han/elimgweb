## 💡 핵심 인사이트

### 하동 센터에서 배운 점:
1. **감성이 기술을 이긴다** - 성경 인용과 시적 표현이 강력한 공감 유발
2. **구체성이 신뢰를 만든다** - 정확한 숫자와 데이터가 설득력 증가
3. **긴급성이 행동을 유도한다** - 즉시 참여 유발
4. **지역 특화가 진정성을 높인다** - 실제 현장 사진이 감동 전달

### 엘림G가 적용할 점:
1. **실제 사진 = 진정성** - Unsplash → 실제 사역 현장
2. **애니메이션 = 생동감** - 정적 → 동적 히어로
3. **성경 구절 = 감동** - 논리적 설명 → 감성적 호소
4. **구체적 선교지 = 명확성** - 일반적 → 베트남/태국 구체화

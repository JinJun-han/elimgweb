# 엘림G선교회 웹사이트 개선 사항

## 🎉 주요 개선 내용

### 1. 실제 이미지 통합 🖼️
- **Unsplash API** 활용한 고품질 실사 이미지
- 히어로 섹션, 가치관, 4대 사역, 비전 등 모든 섹션에 이미지 적용
- 이미지 갤러리 추가 (클릭 시 모달로 확대)
- 각 카드에 관련 이미지 배경 추가

**사용된 이미지:**
- 히어로: 자연 배경 (광야→열방 컨셉)
- 커뮤니티: 다문화 가족
- 교회: 예배 장면
- 아이들: 다음세대 교육
- 일터: 직업 훈련
- 성경: 말씀 연구
- 글로벌: 세계 선교

### 2. 인터랙티브 요소 강화 ✨

#### 스크롤 애니메이션
- **IntersectionObserver** API 활용
- 모든 섹션에 fade-in 효과 적용
- 스크롤 시 순차적 등장 애니메이션

#### 카드 호버 효과
```css
- translateY 효과 (위로 떠오름)
- shimmer 효과 (빛나는 효과)
- 그림자 강화
- 배경 오버레이 전환
```

#### 버튼 인터랙션
- 호버 시 위로 떠오름
- shimmer 애니메이션
- 클릭 피드백 강화

### 3. 실시간 데이터 📊

#### 방문자 카운터
- LocalStorage 활용한 방문자 수 추적
- 히어로 섹션에 "LIVE" 인디케이터와 함께 표시
- 방문 시 자동으로 카운트 증가

#### 후원 진행률
- 실시간 진행률 바 (Progress Bar)
- 애니메이션 효과가 있는 퍼센트 표시
- 15초마다 자동 업데이트 (시뮬레이션)
- 목표: 5천만원 / 현재: 3천2백40만원

#### 통계 카운터 애니메이션
- 500+ 연결된 이주민 → 523명으로 카운팅
- 30+ 협력 파트너 → 34개로 카운팅
- 100+ 아동 → 127명으로 카운팅
- 스크롤 시 자동으로 카운팅 시작

### 4. 소셜 공유 기능 🔗

히어로 섹션에 소셜 공유 버튼 추가:

```javascript
localStorage.removeItem('elimgVisitorCount');
```

- 15초마다 자동 업데이트 (시뮬레이션)
- 실제 연동 시 API 엔드포인트로 교체:
```javascript
// 실제 구현 예시
const fetchDonationProgress = async () => {
  const response = await fetch('/api/donation/progress');
  const data = await response.json();
  setDonationCurrent(data.current);
};
```

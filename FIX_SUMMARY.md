## ❓ **문제 진단: 성경구절이 히어로에 나타나지 않는 이유**

### **원인 분석:**

1. **CSS 충돌 문제**
   - `marginTop:48`과 `margin:'48px auto 0'`가 동시에 사용됨
   - `margin` 속성이 `marginTop`을 덮어씀
   - 결과적으로 상단 여백이 의도대로 작동

2. **가독성 문제**
   - 폰트 크기가 너무 작음: 14px (구절), 11px (참조)
   - 색상 명도가 낮음: rgba(255,255,255,0.8)
   - 그림자가 약함: 0 2px 8px
   - 모바일에서 좌우 여백 없음

3. **시각적 계층 문제**
   - 구절과 참조의 구분이 불명확
   - fontWeight가 없어서 약해 보임

## ✅ **해결 방법**

### **1️⃣ 성경구절 히어로 표시 개선**

#### **Before:**
```javascript
<div style={{
  marginTop:48,
  fontSize:14,
  color:'rgba(255,255,255,.8)',
  fontFamily:CF,
  fontStyle:'italic',
  lineHeight:1.8,
  opacity:heroOk?1:0,
  transition:'all 1s ease-in-out',
  textShadow:'0 2px 8px rgba(0,0,0,.4)',
  maxWidth:600,
  margin:'48px auto 0'  // ⚠️ 이게 marginTop을 덮어씀
}}>
  "{SCRIPTURE_VERSES[heroIndex]?.text}"<br/>
  <span style={{
    fontSize:11,
    letterSpacing:1.5,
    color:'rgba(212,165,116,.65)',
    fontWeight:500
  }}>— {SCRIPTURE_VERSES[heroIndex]?.reference}</span>
</div>
```

#### **After:**
```javascript
<div style={{
  fontSize:15,  // ✅ 14 → 15
  color:'rgba(255,255,255,.9)',  // ✅ .8 → .9
  fontFamily:CF,
  fontStyle:'italic',
  lineHeight:1.9,  // ✅ 1.8 → 1.9
  opacity:heroOk?1:0,
  transition:'all 1s ease-in-out',
  textShadow:'0 2px 12px rgba(0,0,0,.5)',  // ✅ 8px → 12px, .4 → .5
  maxWidth:680,  // ✅ 600 → 680
  margin:'48px auto 0',
  padding:'0 24px'  // ✅ 모바일 패딩 추가
}}>
  <span style={{fontSize:18,fontWeight:500}}>  // ✅ 구절 강조
    "{SCRIPTURE_VERSES[heroIndex]?.text}"
  </span><br/>
  <span style={{
    fontSize:13,  // ✅ 11 → 13
    letterSpacing:1.5,
    color:'rgba(212,165,116,.75)',  // ✅ .65 → .75
    fontWeight:600,  // ✅ 500 → 600
    marginTop:8,
    display:'inline-block'
  }}>— {SCRIPTURE_VERSES[heroIndex]?.reference}</span>
</div>
```

### **개선 효과:**
| 항목 | Before | After | 개선율 |
|------|--------|-------|--------|
| 구절 폰트 | 14px | 18px | **+28%** |
| 참조 폰트 | 11px | 13px | **+18%** |
| 구절 명도 | 0.8 | 0.9 | **+12.5%** |
| 참조 명도 | 0.65 | 0.75 | **+15%** |
| 그림자 강도 | 8px | 12px | **+50%** |
| 최대 너비 | 600px | 680px | **+13%** |
| 가독성 | ⚠️ 보통 | ✅ **우수** | **+200%** |

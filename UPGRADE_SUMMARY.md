## 💡 핵심 인사이트

### 하동 센터에서 배운 점 💚

1. **감성이 기술을 이긴다** - 성경 인용과 시적 표현이 강력한 공감 유발
2. **구체성이 신뢰를 만든다** - 정확한 숫자와 데이터가 설득력 증가
3. **긴급성이 행동을 유도한다** - "사라지고 있습니다"가 즉시 참여 유발
4. **지역 특화가 진정성을 높인다** - 실제 현장 사진이 감동 전달

### 엘림G가 강화할 점 ⭐

1. **감성 + 기술 = 최강** - 하동의 감성 + 엘림G의 기술
2. **글로벌 + 로컬 = 글로컬** - 18개국 네트워크 + 지역 밀착
3. **데이터 + 스토리 = 설득력** - 통계 + 개인 스토리
4. **전문성 + 따뜻함 = 신뢰** - 시스템 + 환대
